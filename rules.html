1O1C NETWORK RULES

Core Regulations & Administration Directives

01. Use Appropriate Channels

Keep discussions relevant to their designated channels. If a game does not have its own section, notify administration instead of posting elsewhere.

02. No Harassment

Any form of harassment, targeted cyberbullying, or intentional server-wide toxicity directed against others will not be tolerated.

03. No Spamming

Avoid excessive messages, including wall-text blocks, emoji floods, caps lock spam, or repeating media loops inside chat systems.

04. Respect Decisions

Follow instructions from core administration staff. Arguing over moderation rulings or wasting processing resources is strictly restricted.

05. Zero Doxxing Protocol

Sharing or threatening to reveal personal real-life data, locations, or identities will result in an immediate network-wide blacklist execution.

06. Profile Restrictions

User profile assets, badges, and server nicknames must remain clean, functional, and free of explicit or suggestive materials.

Enforcement Matrices

  • 1st Level: Formal Structural Warning
  • 2nd Level: Temporary System Communications Mute
  • 3rd Level: Temporary Network Session Suspension
  • Severe Breach: Immediate, Permanent Hardware Blacklist

Core Directives (Staff)

  • OP LOCK: Default root operator commands are fully contained.
  • NO GOD MODE: Server physics and threat variables apply equally.
  • NETWORK BOUND: Admins shall not link with untrusted indices.
